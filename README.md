# NIST CSF Security Incident Response

## Project Overview
This project demonstrates how the NIST Cybersecurity Framework can be used to respond to a security incident.

## Scenario
A DoS attack was detected in the system. An incoming flood of ICMP packets that stopped network services triggered a security investigation

## Framework Used
This project follows the five functions of the NIST Cybersecurity Framework:
- Identify
- Protect
- Detect
- Respond
- Recover

## Investigation Summary
The team found that the disruption was caused by a dustributed denial of of services (DDoS) attack through a flood of incoming ICMP packets

## Response Actions
- Isolate affected systems
- Anayze network logs
